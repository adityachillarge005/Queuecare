# QueueCare

QueueCare is a web-based Hospital Queue and Appointment Management System built with Django. It helps hospitals manage patient registration, appointments, and queue flow in a simple and organized way.

## Features
- Patient registration and profile management
- Appointment scheduling
- Queue management for hospital operations
- Clean Django-based web interface
- SQLite database support

## Tech Stack
- Python
- Django
- SQLite
- HTML/CSS
- Bootstrap
- Django REST Framework (planned)

## Project Description
QueueCare is designed to reduce waiting times and improve hospital workflow by replacing manual queue and appointment processes with a digital system. Administrators, doctors, and patients can work with a centralized structure for better coordination.

## Installation
1. Clone the repository
2. Create and activate a virtual environment
3. Install dependencies
4. Run database migrations
5. Start the development server

Example:
```bash
python manage.py migrate
python manage.py runserver
```

## Usage
Open your browser and navigate to:
```bash
http://127.0.0.1:8000/
```
